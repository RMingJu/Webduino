# Webduino


--------------前言--------------

這是一個網頁工程師與IOT的奇遇......

好啦，我得先承認自己是大學是念電子系，對於微處理機並不陌生，
某天家裡的電燈開關壞了，剛好看到桌子上的Arduino想將家裡的電燈開關都改成可以遠端遙控，
在Google ESP8266晶片來做Arduino連線時，剛好在IT邦幫忙的鐵人賽看到有前輩分享了Webduino，
頓時覺得有趣就下訂了......也就有了這個專案。


webduino其實就是能利用JS來撰寫韌體，讓不懂C語言的網頁工程師也可以輕鬆操作，
重點也不需要接線燒錄可以粉方便的開發。
